## 準備

### Visual Studio 2017
- [ここ](https://www.visualstudio.com/ja/downloads/?wt.mc_id=o~msft~mscom~jpvs2015&rr=https%3A%2F%2Fwww.microsoft.com%2Fja-jp%2Fdev%2Fproducts%2Fvisual-studio-2015.aspx%3FSearch%3Dtrue)
からVisual Studio 2017 Communityをダウンロード。
- exeを起動し、"ユニバーサルWindowsプラットフォーム開発"、"C++によるデスクトップ開発"にチェックを入れてインストール。
- 必要に応じて再起動。

### smail
- [ここ](https://www.picolix.jp/page3.html)から"smail-v4.20.zip"をダウンロード。
- 適当なフォルダに解凍。(例：C:\smail\smail-v4.191)
- 解凍したフォルダ（C:\smail\smail-v4.191）をシステム環境変数に追加。([参考サイト](http://next.matrix.jp/config-path-win7.html))

### felicalib
- [ここ](http://felicalib.tmurakam.org/download.html)から"felicalib-0.4.2.zip"をダウンロード。
- 適当なフォルダに解凍。
